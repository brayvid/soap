# 🚀 SOAP – Critical Next Steps (April 2025)

## 🎯 Phase 1: Launch + Traction (0–4 weeks)
- [ ] **Release Alpha in 1 Test Region**
  - Choose target city or campus
  - Preload key local politicians
  - Seed usage via personal network, Reddit, local orgs

- [ ] **Add Basic User Identity**
  - Pseudonymous account creation (email or OAuth)
  - Store ZIP or general location
  - Enable undo/edit functionality

- [ ] **Track and Normalize Word Submissions**
  - Tag words with metadata (date, location, user ID)
  - Normalize similar words (e.g., “lying”, “liar”, “lies”)
  - Prepare data for future NLP and analytics

---

## 🗳 Phase 2: Product + Trust Infrastructure (4–8 weeks)
- [ ] **Build Localized Public Dashboard**
  - MVP view by city/region
  - Show top 5–10 words per official
  - Optional: Weekly or monthly word shifts

- [ ] **Draft and Publish “Civic Data Ethos” Page**
  - Explain privacy, moderation, no-tracking policy
  - Describe ranked choice mechanics
  - Position SOAP as a public utility

- [ ] **Incorporate as Public Benefit Corp (or Nonprofit)**
  - Choose legal structure aligned with funding strategy
  - Handle IP ownership and contributor agreements

---

## 📈 Phase 3: Growth + Sustainability (8–16 weeks)
- [ ] **Exportable Data Products**
  - CSV export of politician + word cloud data
  - Add timestamps for trend analysis
  - Launch basic API for partners

- [ ] **Fundraising + Partnerships**
  - Prepare pitch for Fast Forward, Knight Foundation, etc.
  - Reach out to journalists and university researchers
  - Create a 1-pager or deck + simple demo link

- [ ] **Launch First Regional Sentiment Report**
  - “Top 10 Words for [City]'s Mayor – Last 30 Days”
  - Include charts, word cloud, and submission count
  - Share with press and local civic groups

---

## 🧼 Ongoing
- Moderate flagged words (manual or auto)
- Monitor for usage spikes or spam
- Update README and changelog
